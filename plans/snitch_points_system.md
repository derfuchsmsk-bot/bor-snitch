# Система "Snitch Points" и Ранги

## 1. Классификация проступков (Snitch Acts)

AI будет анализировать поведение и выбирать одну из категорий для начисления очков.

| Категория | Очки | Описание |
| :--- | :--- | :--- |
| **Нытье (Whining)** | **10** | Жалобы на жизнь, работу, погоду или других участников. |
| **Духота (Stiffness)** | **15** | Занудство, придирки к словам, пассивная агрессия, порча веселья. |
| **Кринж (Cringe)** | **20** | Неудачные шутки, странное поведение, "испанский стыд". |
| **Токсичность (Toxicity)** | **25** | Прямые оскорбления, необоснованная агрессия, грубость. |
| **Предательство (Betrayal)** | **50** | "Слив" инфы, подстава друга, нарушение договоренностей. Самый тяжкий грех. |

## 2. Система Званий (Тюремная Иерархия / Prison Caste)
*Ироничная иерархия: чем больше очков, тем "ниже" статус (но выше ранг Снитча).*

| Очки | Звание (RU) | Описание |
| :--- | :--- | :--- |
| **0 - 99** | **Порядочный** | Еще не зашкварился. Молчит в тряпочку. |
| **100 - 499** | **Шнырь** | На побегушках. Мелкий стукачок. |
| **500 - 999** | **Козёл** | Активист, сотрудничает с администрацией. |
| **1000 - 2499** | **Обиженный** | Его место у параши, но он все равно пишет. |
| **2500+** | **Масть Проткнутая** | Высшая степень падения. Король снитчей. |

## 3. Интеграция

### AI Prompt Update
Системный промпт будет обновлен, чтобы возвращать JSON вида:
```json
{
  "user_id": 12345,
  "username": "ivan",
  "category": "Toxicity",
  "points": 25,
  "daily_title": "Герцог Духоты", // Ежедневный смешной титул
  "reason": "..."
}
```

### DB Schema Update
В `user_stats` добавляем поля:
*   `total_points`: int (accumulated)
*   `current_rank`: string
